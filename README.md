# ChatGPT-Mods

**Браузерное расширение, которое добавляет в интерфейс ChatGPT то, чего там не хватает.**

Навигация по сообщениям · Сплит-экран · Сохранение временных чатов · Показ прикреплённых файлов

---

## Возможности

### 🗂 Навигация по вопросам
Боковая панель с маркерами всех ваших сообщений. Кликните — прокрутится к нужному месту. Наведите курсор — увидите полный текст.

### ↔️ Сплит-экран
Два чата рядом на одном экране с перетаскиваемым разделителем. Удобно для сравнения ответов или параллельной работы в двух контекстах.

### 💾 Сохранение временного чата
Кнопка **«Сохранить»** в шапке временного чата. Собирает историю диалога и переносит её в новый обычный чат — так разговор не потеряется.

> Это перенос контекста через структурированный промпт, а не серверная конвертация чата.

### 📎 Список прикреплённых файлов
Компактный индикатор с тултипом, если к сообщению прикреплено несколько файлов.

## Установка

> Расширение устанавливается вручную как unpacked extension (Chrome / Chromium).

```bash
git clone https://github.com/medvedeff-true/ChatGPT-Mods.git
```

1. Откройте `chrome://extensions`
2. Включите **Режим разработчика**
3. Нажмите **Загрузить распакованное расширение**
4. Выберите папку репозитория
5. Откройте `https://chatgpt.com/`

**Обновление:** после изменения файлов нажмите **Reload** на странице расширений и обновите вкладку ChatGPT.

## Приватность

Расширение работает полностью локально:
- не обращается к сторонним API
- не собирает аналитику
- не отправляет данные куда-либо

Разрешение `storage` используется только для временной передачи данных между вкладками при сохранении временного чата.
